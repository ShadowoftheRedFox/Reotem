Présentation

Bienvenu(e) sur le module administration.

Parmis les modules ci-dessus, vous pouvez:

  • Logs: Vous êtes actuellement dedans, permet de généré les rapports du site ou de récupérer les différentes logs.
  • Utilisateurs: Liste des utilisateurs. Sélectionnez en un pour pouvoir le modifier dans le module modification.
  • Modification: Permet de modifier l'utilisateur selectionné dans la liste.
  • Effacement: Liste des demandes de suppressions d'objets.
